  ## FAQ                                                                                                    
                                                                                                            
  **How do I subscribe?**                                                                                   
  Open the app, tap Settings, and tap "Subscribe for $1.99/month"                                           
                                                                                                            
  **Can I use my own API key?**                                                                             
  Yes! In Settings, choose "Use my own key" and enter your OpenAI or Anthropic API key.                     
                                                                                                            
  **What AI models are used?**                                                                              
  Gestalt uses Claude AI for subscriptions, or your choice of Claude or OpenAI when using your own API key.
